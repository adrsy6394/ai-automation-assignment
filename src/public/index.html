Architecture & Anti-Spam Strategy

🔒 Session Management

Authentication sessions are tracked by browser context cookies. Once registration completes, cookies are saved to cookies/${username}_session.json and re-injected automatically for subsequent actions, avoiding repeated login checks.

🤖 CAPTCHA & Security

Reddit implements CAPTCHA challenge screens. When automated browser instances detect a CAPTCHA iframe on registration, the server gracefully intercepts the event, aborts gracefully, and returns a CAPTCHA detection warning block.

⏳ Human-like Delay Timings

Anti-bot detection is mitigated by interleaving randomized delays (e.g. 1s - 5s) between browser events, focus elements, page navigation load settling, and typing keystroke timing intervals (80ms - 120ms).
